# Instruções para executar o projeto:

## Clone o repositório na sua máquina:

```sh
https://github.com/ThiagoAbreuBR/Challenge-Front-End-AluraGeek.git
```
## Dentro da pasta do projeto, instale as bibliotecas:

```sh
$ npm install
```
## Execute o projeto:

```sh
$ json-server db.json
```
